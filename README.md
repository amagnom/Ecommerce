# Ecommerce WebStore


## Descrição

O projeto é um e-commerce de roupas desenvolvido no ambiente .net. Ele oferece uma plataforma para os usuários navegarem pelos produtos, adicionarem itens ao carrinho, realizarem pedidos e muito mais.
